# Mapsgeo
## Posicionamiento de coordenadas en mapa con marcadores personalizados
Mediante direcciones y coordenadas listadas en una hoja de cálculo se posicionan en el mapa todos los puntos, personalizando los marcadores y la información que se muestra.
